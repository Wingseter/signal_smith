Signal Smith is a Korean stock auto-trading platform with FastAPI backend, Redis, Celery workers/beat, and React+TypeScript frontend. Core flow: news/quant triggers -> AI Council meeting (Gemini/GPT/Claude roles) -> TradingSignal persistence -> auto/manual execution via Kiwoom REST API. Backend contains large orchestration modules (notably council/orchestrator.py and services/tasks/signal_tasks.py). Docker Compose runs redis, backend, celery worker, celery beat, frontend.
